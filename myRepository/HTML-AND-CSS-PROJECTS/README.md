# HTML and CSS Projects

This repository contains a collection of projects created using HTML and CSS, completed as part of a web development course.

---

## Projects

### 1. Academy Cinema

A website for a fictional movie theater. It features a "Now Playing" section, ticket purchasing functionality, and a contact form.
[View Project](./Academy-Cinema/)

### 2. Creating CSS Effects

This project demonstrates various CSS effects on images, including overlays and zoom effects on hover.
[View Project](./Creating-CSS-Effects/)

### 3. Grace Hopper Dedication Page

A single-page website dedicated to the life and achievements of computer science pioneer Grace Hopper.
[View Project](./HTMLCSS_Live%20Coding/)

### 4. Introduction to Bootstrap

A simple project demonstrating the basic features of the Bootstrap framework, such as the grid system.
[View Project](./Introduction-to-Bootstrap/)

### 5. One-Page Website

A single-page website with an anime theme, featuring a navigation bar, image gallery, and contact form.
[View Project](./One-Page%20Website/)

### 6. Simple Recipes

A website for a fictional recipe site called "Simple Recipes." It includes a featured recipe, a collection of recipe cards with flip effects, and a contact form.
[View Project](./Simple-Recipes/)

### 7. Student Portfolio

A personal portfolio website for a student named Andy C. Liang. It includes an about section, a link to their GitHub, and a contact form.
[View Project](./Student_Portfolio/)

### 8. The Pet Shop Website

A simple, static website for a fictional pet shop. It includes a gallery of pets and a contact page.
[View Project](./The-Pet-Shop-Website/)
