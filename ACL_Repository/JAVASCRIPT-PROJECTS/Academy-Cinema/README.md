# Academy Cinemas

This project is a website for a fictional movie theater called Academy Cinemas.

## Features

*   **Now Playing:** Displays a list of movies currently playing, including posters, descriptions, and showtimes.
*   **Ticket Purchasing:** Allows users to select a movie, showtime, and number of tickets to purchase.
*   **Contact Form:** A form for users to get in touch with the cinema.
*   **Responsive Design:** The website is built with Bootstrap, making it responsive and accessible on different devices.

## Technologies Used

*   HTML
*   CSS
*   Bootstrap
