# The Pet Shop Website

This project is a simple, static website for a fictional pet shop. It is created using HTML and CSS.

## Features

*   **Home Page:** Displays a welcome message and a gallery of pets available at the shop.
*   **Contact Page:** Provides contact information for the pet shop.
*   **Pet Sections:** Includes sections for different types of pets like cats, dogs, rabbits, etc.
*   **Embedded Video:** Features an embedded YouTube video.

## Technologies Used

*   HTML
*   CSS
