# Student Portfolio

This project is a personal portfolio website for Andy C. Liang.

## Sections

*   **Home:** A hero section with a quote and a brief introduction.
*   **About:** Information about Andy's background and interests.
*   **GitHub:** A link to Andy's GitHub profile to showcase coding projects.
*   **Contact:** A contact form for reaching out, accompanied by an embedded YouTube video.

## Technologies Used

*   HTML
*   CSS
*   JavaScript (for the YouTube video player)
