# One-Page Anime Website

This project is a single-page website with a theme centered around anime.

## Features

*   **Navigation:** A navigation bar for easy access to different sections of the page.
*   **Hero Section:** A welcoming hero section with a title and a call-to-action button.
*   **About Section:** A brief introduction to the world of anime.
*   **Image Gallery:** A gallery showcasing images of popular anime characters.
*   **Embedded Video:** A section with an embedded YouTube video.
*   **Contact Form:** A contact form for users to send messages.

## Technologies Used

*   HTML
*   CSS
