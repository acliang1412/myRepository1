# Introduction to Bootstrap

This project is a basic introduction to using the Bootstrap framework.

## Description

The `index.html` file demonstrates how to:
*   Include the Bootstrap CSS framework from a CDN.
*   Use the Bootstrap grid system (`container`, `row`, `col`) to arrange elements on the page.
*   Display images within the grid.

## Technologies Used

*   HTML
*   Bootstrap
