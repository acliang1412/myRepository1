# Grace Hopper Dedication Page

This project is a single-page website dedicated to the life and achievements of Grace Hopper.

## Content

*   **Biography:** A short biography of Grace Hopper.
*   **Sections:** Includes sections for her early life, career, retirement, and post-retirement.
*   **Navigation:** A navigation bar allows for easy jumping to different sections of the page.

## Technologies Used

*   HTML
*   CSS
