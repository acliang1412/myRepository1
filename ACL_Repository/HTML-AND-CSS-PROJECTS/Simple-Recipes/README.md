# Simple Recipes

This project is a website for a fictional recipe site called "Simple Recipes".

## Features

*   **Featured Recipe:** A prominently displayed featured recipe.
*   **Recipe Cards:** A collection of recipes presented in a card format.
*   **Flip Effect:** The recipe cards have a flip effect on hover, revealing more details like prep time and difficulty level.
*   **About Us Section:** Information about the "Simple Recipes" website.
*   **Contact Form:** A form for users to contact the site.

## Technologies Used

*   HTML
*   CSS
